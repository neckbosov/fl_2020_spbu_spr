# Дополнительное дз

1. `S -> a S1 c | a S2 cc`

   `S1 -> a S1 с | b`

   `S2 -> a S2 cc | d`

2. `FIRST(S) = {a}`, отсюда в ячейке `(S, a)`
   управляющей таблицы будет лежать два правила, поэтому данная грамматика не является LL(1).

3. `S -> A K | A L`

   `K -> S1 C`

   `L -> S2 Q`

   `S1 -> A K | b`

   `S2 -> A L | d`

   `Q -> C C`

   `A -> a`

   `B -> b`

   `C -> c`

   Выводы слов в приложенных картинках.
