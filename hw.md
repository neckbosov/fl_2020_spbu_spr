# Дополнительное дз

1. `S -> S1 | S2`

   `S1 -> a S1 с | b`

   `S2 -> a S2 cc | d`

2. `FIRST(S1) = {a, b}`,  `FIRST(S2) = {a, d}`, отсюда `FIRST(S) = {a, b ,c}` и в ячейке `(S, a)`
   управляющей таблицы будет лежать `S1` и `S2`, поэтому данная грамматика не является LL(1).

3. `S -> A P | b | A R | d`

   `S1 -> A P | b`

   `P -> S1 С`

   `S2 -> A R | d`

   `R -> S2 Q`

   `Q -> C C`

   `A -> a`

   `B -> b`

   `C -> c`

   Выводы слов в приложенных картинках.
